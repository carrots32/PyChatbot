

Below is an example of what should be in listQA.txt

NOTE: The gaps and spaces are important!
NOTE: even after the final ; in series there must be a space.
NOTE: All questions MUST be in all lowercase!
NOTE: BOTTOM has priority. ie if userInput == "question 19 question2 question11?", answer will be 'answerX' or 'answerY'
                SOMETIMES!!! Order of x where question[x] too. 
                ie. Q1 before Q10 because Q! has x=1, Q10 has x=5
NOTE: use '%n' for a linebreak
